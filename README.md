# zkWormholesWifToken


### install nargo v0.25.0   
([text](https://noir-lang.org/docs/v0.25.0/getting_started/installation/))  
```
curl -L https://raw.githubusercontent.com/noir-lang/noirup/main/install | bash;

noirup --version v0.25.0;
```

### compile circuit

```
cd circuit
nargo compile
```


### run website

```
cd website
yarn install
yarn run dev

```
